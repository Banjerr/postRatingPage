Post Rating System

 Download Here

WP plugin that allows you to add a rating system (for logged in users) to posts/custom posts/pages/etc. (default is set to a custom post type, which is also created by the plugin on activation)

How To Use

  • Install (upload to your WP plugins directory and activate)
  • In your template just call the `echo` the function like this...
    • echo generateRatingHTML(get_the_ID());
  • override whatever styles you need to
  • rate some stuff
